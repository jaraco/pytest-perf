Added support for measuring import latency. A benchmark function whose name contains ``import_time`` (e.g. ``check_import_time``) is timed with ``python -X importtime`` instead of ``timeit``, which cannot measure imports because ``sys.modules`` caches them after the first loop. On interpreters that don't emit an ``-X importtime`` trace (e.g. PyPy), such benchmarks are skipped.
